# My configs

## Description

Pack of my configs for vim, tmux, zsh, termite

## Installation

```sh
$ cd ~
$ git clone --recursive https://github.com/dmitry64/shell-configs.git
$ cd shell-configs
$ ./install.sh
```

### Required packages:
* zsh
* tmux
* termite
* vim
* terminus-font
* powerline
* oh-my-zsh
